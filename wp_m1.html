World Wide Web (WWW) and Its Evolution

The World Wide Web (WWW) is a system of interlinked hypertext documents accessed through the Internet using web browsers. 
It was invented by Tim Berners-Lee in 1989 and became publicly accessible in 1991. 
The WWW operates using the Hypertext Transfer Protocol (HTTP) and is built on technologies like HTML, URLs, and web browsers.

Evolution of the Web:

1. Web 1.0 (Static Web) (1991 – 2004)
   - Consisted of static web pages.
   - Users could only read the information; there was no interaction.
   - No multimedia elements (images, videos, dynamic content).

2. Web 2.0 (Dynamic and Social Web) (2004 – Present)
   - Interactive websites with user-generated content (e.g., blogs, wikis, social media).
   - Technologies like JavaScript, AJAX, and CSS enabled better user experience.
   - Platforms like Facebook, YouTube, and Twitter emerged.

3. Web 3.0 (Semantic Web & AI-based Web) (Future)
   - Focuses on machine learning, AI, and decentralized data.
   - Uses blockchain and smart contracts (e.g., cryptocurrency and NFTs).
   - More personalization and automation (e.g., AI-driven assistants).

E-mail (Electronic Mail)

E-mail is a method of exchanging digital messages over the Internet. It consists of a sender, recipient, subject, body, and attachments.

Key Protocols Used in E-mail:
1. SMTP (Simple Mail Transfer Protocol): Used to send emails.
2. IMAP (Internet Message Access Protocol): Used to receive emails, keeps emails on the server.
3. POP3 (Post Office Protocol 3): Used to receive emails but downloads and removes them from the server.

Popular E-mail Services:
- Gmail
- Yahoo Mail
- Outlook
- ProtonMail

Telnet

Telnet (TELecommunication NETwork) is a protocol used to remotely access computers over a network. 
It allows users to execute commands on a remote system as if they were physically present.

Features of Telnet:
- Uses port 23 by default.
- Provides a command-line interface to access remote computers.
- Security Concern: Data transmission is not encrypted, so it has been largely replaced by SSH (Secure Shell).

FTP (File Transfer Protocol)

FTP is a standard network protocol used to transfer files between computers.

Types of FTP Connections:
1. Active FTP: The server actively connects to the client’s data port.
2. Passive FTP: The client requests the server to open a data port for file transfer.

Common FTP Commands:
- get filename → Downloads a file.
- put filename → Uploads a file.
- ls → Lists files in the directory.

Secure Alternatives to FTP:
- SFTP (Secure FTP): Uses SSH to encrypt data.
- FTPS (FTP Secure): Uses SSL/TLS encryption.

E-Commerce (Electronic Commerce)

E-Commerce refers to buying and selling goods and services over the Internet.

Types of E-Commerce:
1. B2B (Business to Business): Transactions between businesses (e.g., Alibaba, SAP).
2. B2C (Business to Consumer): Businesses selling directly to consumers (e.g., Amazon, Flipkart).
3. C2C (Consumer to Consumer): Transactions between individuals (e.g., eBay, OLX).
4. C2B (Consumer to Business): Individuals offer services to businesses (e.g., Fiverr, Upwork).

Popular E-Commerce Payment Methods:
- Credit/Debit Cards
- Digital Wallets (PayPal, Google Pay)
- Cryptocurrencies (Bitcoin, Ethereum)

Cloud Computing

Cloud computing refers to delivering computing services (servers, storage, databases, networking, software) over the Internet.

Types of Cloud Computing:
1. IaaS (Infrastructure as a Service): Provides virtualized computing resources (e.g., AWS, Google Cloud).
2. PaaS (Platform as a Service): Provides platforms for app development (e.g., Heroku, Azure).
3. SaaS (Software as a Service): Provides software applications (e.g., Google Drive, Dropbox).

Benefits of Cloud Computing:
- Scalability (Easy to increase/decrease resources)
- Cost-Effective (No need for physical hardware)
- Accessibility (Access from anywhere)

Video Conferencing

Video conferencing allows real-time video and audio communication over the Internet.

Popular Video Conferencing Platforms:
- Zoom

Advantages of Video Conferencing:
- Reduces travel costs
- Allows remote collaboration

IP Address (Internet Protocol Address)

An IP address is a unique identifier assigned to each device on the Internet.

Types of IP Addresses:
1. IPv4 (Internet Protocol version 4)
- 32-bit address (e.g., 192.168.1.1)
- Limited to 4.3 billion addresses.
2. IPv6 (Internet Protocol version 6)
- 128-bit address (e.g., 2001:db8::ff00:42:8329)
- Supports an almost unlimited number of devices.

Search Engine

A search engine helps users find web pages based on search queries.

Popular Search Engines:
- Google (Most popular)
- Bing (Microsoft’s search engine)
- Yahoo Search
- DuckDuckGo (Privacy-focused)

How Search Engines Work:
1. Crawling: Bots collect data from web pages.
2. Indexing: Organizes pages in a database.
3. Ranking: Uses algorithms to display results based on relevance.
